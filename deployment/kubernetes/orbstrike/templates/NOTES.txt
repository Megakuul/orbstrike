Installed Orbstrike-System!

For the Longhorn Cluster, its required to have *open-iscsi* installed on each node.

You will also need to initialize the REDIS Cluster with following command from inside the cluster:

redis-cli --cluster create \

{{- end }}
--cluster-replicas {{ add .Values.cacheDBReplicas -1 }}
